# Weighted Calisthenics — Added-Load Standards

Added load is expressed as **% of bodyweight ADDED** (on a dip belt / vest / held), for a strict **1–5 rep working set**. The lb column anchors to a **180 lb lifter**. These are practical, coaching-consensus bands tuned for a skill-rank ladder — not StrengthLevel's lifetime-best percentiles (which run much higher; see the reconciliation note).

## Primary table (recommended for the ladder)

| Movement | unit (%bw added) | Beginner | Novice | Intermediate | Advanced | Elite | lb@180 (Beg / Nov / Int / Adv / Elite) | confidence | sources |
|---|---|---|---|---|---|---|---|---|---|
| **Weighted pull-up** | % bw added | 0–10% | 10–25% | 25–50% | 50–75% | 100%+ | +0–18 / +18–45 / +45–90 / +90–135 / +180+ | High | Liftoff, Endura, Quora, BullBar |
| **Weighted chin-up** | % bw added | 0–10% | 15–30% | 30–55% | 55–85% | 100%+ | +0–18 / +27–54 / +54–99 / +99–153 / +180+ | Medium | Quora, StrengthFirst, Liftoff (analog) |
| **Weighted dip** | % bw added | 0–15% | 15–35% | 35–60% | 60–90% | 110%+ | +0–27 / +27–63 / +63–108 / +108–162 / +200+ | High | Liftoff, Endura, Kalibre |
| **Weighted muscle-up** | % bw added | 0% (BW only) | 0–10% | 10–25% | 25–45% | 50%+ | +0 / +0–18 / +18–45 / +45–81 / +90+ | Low–Med | Calisthenics consensus, FitnessVolt |
| **Weighted pistol squat** | % bw added (held) | 0–15% | 15–30% | 30–50% | 50–70% | 90%+ | +0–27 / +27–54 / +54–90 / +90–126 / +162+ | Medium | Endura, Steven Low |

**Reading the Elite cell:** for the upper-body pulls/press, Elite = "a full second bodyweight (or more) added" — the classic +100% bw pull-up milestone. Dip Elite sits slightly higher (~110%+) because the pressing line carries more load. Muscle-up and pistol Elite are lower in %bw terms because the movement itself (explosive transition / single-leg balance) caps loadable weight well before the prime mover fails.

## Reconciliation note — why two number sets exist

There are two distinct "standards" worlds and they disagree by a wide margin. The table above uses **set #1**.

**Set #1 — coaching / %bw benchmarks (used above).** Convergent across Liftoff's prose section, Endura calculators, Quora coach answers, and BullBar:
- Pull-up: Beginner 5–10%, Intermediate 25–50%, Advanced 50–75%, **Elite 100%+** bw added.
- Dip: Beginner ~10% assist→BW, Intermediate ~55%, Advanced ~85%, Elite 100%+ (dip allows more than pull-up).
- Pistol squat (Endura, ×BW added): Beginner <0.32, Novice 0.32–0.48, Intermediate 0.48–0.68, Advanced 0.68–0.92, Elite ≥0.92 — I scaled the ladder *down* from these because Endura's bands describe a held-load 1RM and skew high for a skill-rank context; the table uses 0–15 / 15–30 / 30–50 / 50–70 / 90%+.

**Set #2 — StrengthLevel lifetime-best percentiles (NOT used; shown for cross-check).** Sourced via Liftoff's blog tables (which republish StrengthLevel data, anchored to the ~190 lb average male):
- Weighted **pull-up**: Beginner −28 lb (assisted), Novice +18, Intermediate +75, Advanced +138, **Elite +206 lb**.
- Weighted **dip**: Beginner −17 lb (assisted), Novice +41, Intermediate +110, Advanced +190, **Elite +276 lb**.

At 180 lb, StrengthLevel "Elite" pull-up (+206) ≈ **115% bw** and "Elite" dip (+276) ≈ **153% bw** — i.e. StrengthLevel's tier names sit roughly one rung "harder" than the coaching benchmarks. StrengthLevel "Advanced" (~+138 / +77% bw pull-up) lines up with the coaching **Elite** threshold (+100% bw). For a motivating in-app ladder, Set #1 is the better fit; reserve Set #2 numbers if you want a true 99th-percentile "Elite" ceiling.

## Per-movement detail

**Weighted pull-up.** Strongest source coverage. The +100% bw added (a full second bodyweight, +180 lb @180) is the universally cited "elite" milestone. Confidence **High**.

**Weighted chin-up.** No dedicated dataset; supinated grip is typically **~5–15% stronger** than the same lifter's pull-up (biceps contribution), so each band is shifted up modestly from the pull-up. Same +100% bw Elite milestone. Confidence **Medium** (interpolated).

**Weighted dip.** Strong pressing line → carries **more added load than pull-ups**, confirmed across sources. Coaching bands: Intermediate ~35–60%, Advanced ~60–90%, Elite 110%+ bw. Liftoff/StrengthLevel absolute numbers run much higher (see Set #2). Confidence **High** on the "dip > pull-up" ordering; **Medium** on exact Elite cutoff.

**Weighted muscle-up.** Weakest data — most "muscle-up standards" online are unweighted (rep counts). Added load is capped by the explosive transition, not max pulling strength, so loadable weight is far lower than a weighted pull-up: a clean **+25% bw muscle-up is genuinely advanced**, +50%+ is elite/competition territory. StrengthLevel's "weighted muscle-up" entries are sparse and near-zero added load (it's tracking the unweighted skill). Confidence **Low–Medium**.

**Weighted pistol squat (held load).** Load held at chest/goblet or in a vest while doing a single-leg full-depth squat. Balance and ankle/knee position cap the load before the quad fails, so practical added load is lower than a barbell squat would predict. Endura tiers (×BW added) anchor the high end; the ladder is scaled modestly down for a skill context. Confidence **Medium**.

## Source agreement / confidence summary

- **Strong agreement:** pull-up and dip %bw ladders; the "+100% bw = elite pull-up" milestone; "dip allows more added load than pull-up." Multiple independent coaching sources converge.
- **The one real disagreement:** absolute-lb tier cutoffs. StrengthLevel/Liftoff percentile tiers (Set #2) are ~1 rung harder than coaching %bw benchmarks (Set #1). This is a methodology gap (lifetime-best 99th-percentile vs. coaching "what's strong"), not a data error — documented above so the ladder author can pick a ceiling deliberately.
- **Weakest evidence:** weighted muscle-up (mostly unweighted data online) and the exact chin-up offset (interpolated from pull-up, no standalone dataset).
- **StrengthLevel direct fetch:** retried multiple times (raw, browser UA, translate.goog proxy) — the standards tables are JS-rendered and not retrievable server-side. Numbers above come from Liftoff's blog, which republishes StrengthLevel's dataset, plus the Endura calculators.
