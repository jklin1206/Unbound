Violet should feel rare, not ambient.

The app palette is mostly near-black, graphite, and bone white. The issue is not the token set itself. The issue is that Home, Skills, and Program use accent violet as the default state color too often, so the screens begin to read as purple-first instead of monochrome-first.

Working Rule

Keep the palette, rebalance the surfaces. Violet should mark active, selected, primary, unlocked, or high-emotion states. It should not be the default tint for every chip, line, card halo, row icon, progress pill, and decorative border.

Proposed Screen Rules

Home

Use violet for the primary start action, active streak/current-day marker, and one key number. Use graphite rows and neutral borders everywhere else. Ember can represent completed training heat.

Skills

Use violet only for current progression, active goal, selected tab, and unlock path. Locked/future nodes should be graphite. Mastered/mythic can use impact violet, but sparingly.

Program

Use violet for today/current block and the Start button. Exercise rows, swaps, weekly cards, and secondary coach actions should default to neutral, cyan, amber, or ember depending on meaning.
