The Movement Library Is The Core

UNBOUND needs one clean movement language for programs, substitutions, skill training, workout logging, rewards, and ranks.

Do rankreal movements users chase
Do not rankrandom strings, warmups, notes, or routine titles
Resolvealiases and variations into canonical movements
Showsmall curated choices, not a database dump

9 Ranks Are The Only Rank Standard

  1. Initiate
  2. Novice
  3. Apprentice
  4. Forged
  5. Veteran
  6. Honed
  7. Vessel
  8. Unbound
  9. Ascendant

The older 5-level skill model is deprecated. It can remain as temporary legacy copy, but it should not determine unlocks, rank standards, or user-facing achievement truth.

Source of truth: SkillTier and future exercise performance rank templates.

Two Rank Systems, One Log

Skill Rank

Answers: can you perform the skill to standard?

  • Strict reps
  • Hold seconds
  • Prerequisites
  • Form ownership
  • No LP rank-up without proof

Exercise Performance Rank

Answers: how strong or capable are you at this movement?

  • e1RM
  • Bodyweight ratio where useful
  • Distance/duration
  • Load carried
  • LP as progress toward next proven standard

One pull-up log can update both a pull-up skill standard and a pull-up performance rank, but the app should display those as separate meanings.

What Gets Ranked

Canonical gym liftsbench, squat, deadlift, leg press, cable rowranked
Bodyweight movementspull-up, push-up, dip, pistol, plankranked
Skillshandstand, muscle-up, plancheskill ranked
Aliasesband-assisted, tempo, negative, machine variantsfeed target rank
Routine templatesMorning Mobility Flow, Zero Limit Protocolnot ranked directly
Warmups/contextcool-down, rest, notesnot ranked

Rank Template Families

Barbell Strength

bench, squat, deadlift, overhead press and close variants

Metric: e1RM / stable bodyweight plus absolute milestones

Machine / Cable Strength

leg press, hack squat, chest press, pulldown, rows, curls, extensions

Metric: best clean working load, reps, and machine-adjusted standard

Bodyweight Rep Skill

pushup, pullup, dip, pistol/shrimp regressions, nordic variants

Metric: strict reps to form standard, no bodyweight scaling for base version

Weighted Bodyweight

weighted pullup, weighted dip, weighted pistol and loaded calisthenics

Metric: added load / stable bodyweight

Hold / Control

planks, hangs, hollow holds, handstand holds, lever holds

Metric: hold seconds with quality flags

Carry / Sled

farmer carry, suitcase carry, sled push, loaded march

Metric: load, distance, duration, posture quality, side parity

Cardio Performance

run, row, bike, walk, swim, stairs, elliptical

Metric: duration, distance, pace/power/calories, HR/RPE context

Mobility / Stretch

hip flexor, hamstring, thoracic, wrist prep, shoulder mobility

Metric: duration, side, consistency, pain-free quality, range ownership

Barbell Strength

Applies to: bench, squat, deadlift, overhead press and close variants

Metric: e1RM / stable bodyweight plus absolute milestones

  1. InitiateTechnique proof
  2. Novice0.45x BW
  3. Apprentice0.65x BW
  4. Forged0.85x BW
  5. Veteran1.00x BW
  6. Honed1.20x BW
  7. Vessel1.40x BW
  8. Unbound1.65x BW
  9. Ascendant1.90x BW

Exact multipliers vary by lift. Bench/squat/deadlift should not share the same numbers, but they share the template shape.

Machine / Cable Strength

Applies to: leg press, hack squat, chest press, pulldown, rows, curls, extensions

Metric: best clean working load, reps, and machine-adjusted standard

  1. InitiateClean setup
  2. NoviceControlled sets
  3. ApprenticeWorking load base
  4. ForgedStrong beginner
  5. VeteranSolid gym strength
  6. HonedAdvanced control
  7. VesselStandout load
  8. UnboundRare load
  9. AscendantElite machine output

Machine stacks vary, so the rank should lean on progression within the exercise plus broad standards, not pretend all machines are identical.

Bodyweight Rep Skill

Applies to: pushup, pullup, dip, pistol/shrimp regressions, nordic variants

Metric: strict reps to form standard, no bodyweight scaling for base version

  1. Initiate1 clean rep
  2. Novice2-3 reps
  3. Apprentice3-5 reps
  4. Forged5 clean reps
  5. Veteran7-8 reps
  6. Honed8-10 reps
  7. Vessel10-12 reps
  8. Unbound12-15 reps
  9. Ascendant15+ or harder variant

This is the model already used by pull-up skill tiers: strict movement standard first.

Weighted Bodyweight

Applies to: weighted pullup, weighted dip, weighted pistol and loaded calisthenics

Metric: added load / stable bodyweight

  1. InitiateLogged weighted rep
  2. Novice0.10x BW
  3. Apprentice0.20x BW
  4. Forged0.35x BW
  5. Veteran0.50x BW
  6. Honed0.65x BW
  7. Vessel0.80x BW
  8. Unbound1.00x BW
  9. Ascendant1.25x BW

This is already present for weighted pull-up and should become the standard for weighted bodyweight work.

Hold / Control

Applies to: planks, hangs, hollow holds, handstand holds, lever holds

Metric: hold seconds with quality flags

  1. Initiate10s
  2. Novice20s
  3. Apprentice30s
  4. Forged45s
  5. Veteran60s
  6. Honed90s
  7. Vessel120s
  8. Unbound150s
  9. Ascendant180s+ or harder shape

Hold ranks need quality requirements. A sloppy 90s hold should not outrank a clean 45s hold.

Carry / Sled

Applies to: farmer carry, suitcase carry, sled push, loaded march

Metric: load, distance, duration, posture quality, side parity

  1. InitiatePattern learned
  2. NoviceLight load
  3. ApprenticeModerate load
  4. ForgedBodyweight-equivalent work
  5. VeteranHeavy repeatable work
  6. HonedLong heavy carry
  7. VesselStandout load
  8. UnboundVery heavy carry
  9. AscendantElite carry standard

Carry ranks need both load and distance/duration. Load alone is too easy to game.

Cardio Performance

Applies to: run, row, bike, walk, swim, stairs, elliptical

Metric: duration, distance, pace/power/calories, HR/RPE context

  1. InitiateComplete short session
  2. NoviceConsistent easy work
  3. ApprenticeBaseline distance
  4. ForgedRespectable pace
  5. VeteranStrong pace
  6. HonedAdvanced pace
  7. VesselStandout benchmark
  8. UnboundVery high benchmark
  9. AscendantElite app benchmark

Cardio needs modality-specific standards. A 5k run rank and 400m row rank should share rank names but not thresholds.

Mobility / Stretch

Applies to: hip flexor, hamstring, thoracic, wrist prep, shoulder mobility

Metric: duration, side, consistency, pain-free quality, range ownership

  1. InitiateExposure
  2. NoviceBasic positions
  3. ApprenticeConsistent practice
  4. ForgedPain-free control
  5. VeteranUsable range
  6. HonedStrong range
  7. VesselAdvanced range
  8. UnboundDurable range
  9. AscendantExceptional range

Mobility ranks should be gentler and slower. They should not reward random long stretching without quality.

AP: Ascension Points

AP replaces LP language. It measures progress toward the next proven standard, but it does not rank the user up by grind alone.

RankThe standard already proven
APProgress toward the next standard
MilestoneIconic achievement badges like 225 bench
TrialOverall rank-up event when readiness is high

Example: Bench Press · Forged · 78 AP toward Veteran · Next Standard: 1.0x BW e1RM.

AI Boundary

The movement library and resolver should make the core app work without AI. AI is optional, structured, and validated by deterministic rules.

Rules Own

  • program generation
  • skill insertion
  • substitutions
  • deloads/travel weeks
  • AP/ranks/trials
  • rest and volume protection

AI Can Help

  • parse messy free text
  • explain why a workout changed
  • summarize progress
  • write trial briefing flavor
  • choose among already-valid options

User Experience Rule

The user should never browse hundreds of exercises by default.

Program slot
Recommended movement
3-6 smart substitutes
Search if needed

Example: Quad-dominant strength slot -> Back Squat -> Front Squat, Hack Squat, Leg Press, Goblet Squat, Smith Squat.

Substitutions Are Slot-Based

Bad

“Pick any exercise from 177 movements.”

Good

“This slot needs a quad-dominant strength movement. Here are options that match your equipment, rank, history, and preferences.”

Substitution groups should use pattern, body part, equipment, difficulty, rank template, and user preference.
